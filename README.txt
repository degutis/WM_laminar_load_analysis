Analysis script for the preprint entitled "Dynamic layer-specific processing in the prefrontal cortex during working memory" by Degutis et al. 2023
The repository contains analysis scripts for the layer-specific univariate and multivariate results.
The data accompanying the analysis script will be uploaded following publication. 
All figures in the preprint can be reproduced using gradAnalysisScript.m
